INFERNO SUPPORT

Expert guides and technical specifications for the Inferno Gaming Mouse.

Product Overview

Sr. no Question Answer
1 What’s in the box?
  • 1 x Inferno Gaming Mouse
  • 1 x Set of Extra Techflon Skates (0.6mm)
  • User Manual
2 What is the Inferno Gaming Mouse? The Inferno is a USB Wired RGB gaming mouse that bridges the gap between professional performance and everyday portability. Weighing in at 75g, designed for long gaming hours.
3 What are the key features of the Inferno?
  • 16,000 DPI Adjustable Sensor
  • 1000Hz polling rate
  • Huano Switches

Quick Setup Guide

Sr. no Question Answer
1 How do I power ON the mouse? The mouse powers on automatically when connected via the cable.
2 How do I connect the mouse? Wired: Plug the cable into the PC USB port.
3 Do I need to install drivers? Basic functionality works without drivers. To customize DPI, buttons, RGB lighting, polling rate, and profiles, installing the Karnage Gaming Mouse software is recommended.
4 How do I reset the mouse? Disconnect the mouse and reconnect the mouse using wired mode and reconfigure settings via software.

Features and Customization

Sr. no Question Answer
1 How can I adjust DPI settings? DPI can be adjusted through the software from 100 to 16,000 DPI. Default DPI steps include 800-1600-2400-3200-6400-12800.
2 What are the programmable buttons used for? The Inferno features programmable keys that can be remapped via software for macros, shortcuts, or in-game actions to enhance gameplay efficiency.
3 How do I control the RGB lighting? Use the dedicated RGB Button on the mouse or customize effects through the Karnage software.

Compatibility

Sr. no Question Answer
