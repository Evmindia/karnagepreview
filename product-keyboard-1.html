Frequently Asked Questions

1. How do I connect the Visage keyboard to my device?

You can connect in three ways:

  • 2.4GHz Wireless: Plug in the USB receiver, turn on the keyboard, and press Fn + 3
  • Bluetooth: Hold Fn + 1 or Fn + 2 for 3 seconds to pair
  • Easily switch between devices using Fn + 1 / 2 / 3
2. How long does the battery last and how do I charge it?

The keyboard features a 2000mAh rechargeable battery that lasts up to 100 hours on a single charge.

Charge it using the Type-C cable with any standard USB port or 5V adapter.

3. Can I customize the RGB lighting?

Yes, you have full control over lighting:

  • Toggle RGB: Fn + Backspace
  • Change effects: Fn + Enter
  • Adjust brightness: Fn + ↑ / ↓
  • Control speed: Fn + → / ←
  • Set static color: Fn + |/
4. Is the keyboard compatible with multiple devices and operating systems?

Absolutely. The keyboard supports Windows, macOS, iOS, and Android.

You can also quickly switch between devices and OS modes using simple Fn shortcuts.
